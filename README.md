# Elevate_labs-Task4


## Objective
To design an interactive dashboard for business stakeholders that provides insights into sales performance, profitability, and growth trends.

## Tool Used
- Power BI

## Dataset
- Sales  Dataset sourced from Kaggle

## Dashboard Features
- KPI cards for Total Sales, Total Profit, Profit Margin, and Products Sold
- Interactive slicers for Date, City, Product Name, and Channel
- Time-series analysis comparing current year and previous year sales
- Product-wise, customer-wise, channel-wise, and city-wise analysis
- Consistent color theme for better readability and usability

## Deliverables
- Interactive Power BI Dashboard
- PPT summary explaining objectives, KPIs, visuals, and insights

## Outcome
The dashboard enables stakeholders to monitor business performance efficiently and supports data-driven decision-making.
